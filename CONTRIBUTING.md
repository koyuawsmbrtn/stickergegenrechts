# Contributing to Sticker gegen Rechts

Thank you for your interest in contributing to "Sticker gegen Rechts"! This guide will help you get started.

## How to Contribute

1. Fork the repository
2. Create a new branch for your contribution
3. Make your changes
4. Submit a pull request

## Guidelines

- Use appropriate file formats (SVG preferred for vector graphics)
- Keep the code clean and readable
- Follow the project's code of conduct
- Make sure your changes are well-tested
- Document your changes
- Make sure your changes are compatible with the project's license

## Code of Conduct

- Be respectful and inclusive
- No hate speech or discriminatory content
- Follow anti-fascist principles
- Help maintain a welcoming environment

## Questions?

If you have any questions, feel free to open an issue or contact the maintainers.
